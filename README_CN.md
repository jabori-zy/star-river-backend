# Star River Backend

[English](./README.md) | [中文](./README_CN.md)

Star River Backend 是一个基于 Rust 开发的，以 Workflow 为基建的量化交易后端，采用事件驱动架构设计，保证策略的高性能和高安全性。

## 快速开始

```bash
# 启动服务（默认端口 3100）
cargo run

# 指定端口
cargo run -- --port 3200

# 调试模式
cargo run -- --log-level debug
```
