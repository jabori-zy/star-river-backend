# Star River Backend

[English](./README.md) | [中文](./README_CN.md)

Star River Backend is a workflow-based quantitative trading backend built with Rust, featuring an event-driven architecture that ensures high performance and security for trading strategies.

## Quick Start

```bash
# Start the server (default port 3100)
cargo run

# Specify port
cargo run -- --port 3200

# Debug mode
cargo run -- --log-level debug
```
